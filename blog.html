Important FAQ

1. What is the DOM in JavaScript?

Answer: The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing JavaScript to manipulate, access, and update the content, structure, and styling of the webpage dynamically.

2. How do you select an HTML element using JavaScript?

Answer: You can select an HTML element using methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName(). For example: 

                    const element = document.getElementById("myElement");
                    const element = document.querySelector(".myClass");
3. How do you add an event listener to an element in JavaScript?

Answer: To add an event listener to an element, you can use the addEventListener() method. For example: 

const button = document.getElementById("myButton");
button.addEventListener("click", function() {
alert("Button clicked!");
                    });
4. How do you create and append an HTML element dynamically in JavaScript?

Answer: You can create a new HTML element using document.createElement() and append it to a parent element using appendChild(). For example: 

const newElement = document.createElement("div");
newElement.textContent = "This is a new element";
document.body.appendChild(newElement);
